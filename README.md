# ⚽ Mundial Predictor ML

Proyecto de Machine Learning para predecir partidos y simular un Mundial usando modelos de clasificación.

## 🚀 Características

- Predicción de partidos
- Probabilidades de victoria
- Simulación completa del torneo
- Simulación Monte Carlo
- Probabilidad de campeón
- Dashboard interactivo con Streamlit


## 🤖 Modelos usados

- Regresión Logística
- Random Forest
- XGBoost


## 📊 Variables utilizadas

Datos de equipos:

- Ranking FIFA
- Valor de plantilla
- Jugadores Champions League
- Edad promedio
- Fuerza ofensiva
- Fuerza defensiva
